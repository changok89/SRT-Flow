## What to include in your email
To help us resolve issues quickly, please include:
- Device model and iOS version
- App version
- Steps to reproduce
- Screenshot or screen recording (if possible)

## Common issues
1. Speech recognition is not working
- Check microphone and speech recognition permissions in iOS Settings.
- Make sure internet connectivity is available.

2. Translation is unavailable
- Translation Framework mode requires iOS 18 or later.
- Internet connection is required for translation model usage.

3. File import/export issues
- Ensure Photos/Files access permission is granted.
- Verify enough storage space on device.

## 자주 묻는 문제
1. 음성 인식이 동작하지 않는 경우
- iOS 설정에서 마이크/음성 인식 권한을 확인해주세요.
- 인터넷 연결 상태를 확인해주세요.

2. 번역이 동작하지 않는 경우
- Translation Framework 모드는 iOS 18 이상이 필요합니다.
- 번역 기능 사용 시 인터넷 연결이 필요합니다.

3. 파일 가져오기/내보내기 문제
- 사진/파일 접근 권한을 확인해주세요.
- 기기 저장 공간이 충분한지 확인해주세요.
